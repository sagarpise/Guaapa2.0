Mobikul Merge Category

Create/Link your existing Website Category with products on Mobikul

It is quite a simple yet useful module, which exports all your product's categories to 'Mobikul', and links all the products with their respective categories in 'Mobikul'. If you are planning to run or already running 'Mobikul' in Odoo and you want to categorize your products in 'Mobikul' as well, then 'Mobikul Merge Category Module', is the best product for you.

Create Mobikul Categories on one click

Using this module, you can create all website product's categories in 'Mobikul' with just 1 click.

Link Products to Mobikul Categories

Using this module, you can link all created Website Categories in 'Mobikul' with just 1 click.
