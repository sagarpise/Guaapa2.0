Configuration for api guaapa

Go to technical -> system parameters -> create

Steps

  • CREATE USER:
    add key -> guaapa_user: "USER MLS"
  • CREATE PASSWORD:
    add key -> guaapa_password: "PASSWORD MLS"
  • CREATE NUMBER CLIENT:
    add key -> guaapa_num_client: "NUMBER CLIENT"
  • Generate shipments:
    add key -> guaapa_url_delivery: "URL CREATE"
  • Check shipments:
    add key -> guaapa_url_check_delivery: "URL CHECK"
